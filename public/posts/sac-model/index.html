Create new model on SAC

December 11, 2021 Â· 1 min | Suggest Changes

Let’s build a very simple SAC model using kaggle data on video game sales.

  1. To create a model on SAC, firstly you have to login into your account (please search for trail if necessary). Next click on files, plus icon, and select model.
  1. Now, please click on From a CSV or Excel File
  1. Select Source File

  1. Download source file from Kaggle.

  2. Find vgsales.csv and load into SAC

  1. Click on import
  1. This model is very simple so we don’t have to change anything. Just click on Create Model
  1. Save data model
